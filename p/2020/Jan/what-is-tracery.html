what is tracery

Jan 22, 2020


Tracery is a library that can generate sentances and other things. It reads
from a JSON file with a sentance structure inside it, then it can generate
anything you want! here are some intresting things generated with it:

 my name is Landon and my favorite distribution is Ubuntu MATE
